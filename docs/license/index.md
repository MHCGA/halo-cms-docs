# 版权说明

Halo CMS 知识库的代码与文稿遵循不同的授权模式。请根据以下说明在使用、分发或再创作内容前确认自身义务。

- 仓库中的构建脚本、主题扩展与自动化配置默认采用 [MIT License](https://opensource.org/licenses/MIT)。
- 你可以自由地复制、修改、分发代码，亦可用于商业目的。
- 使用 MIT 代码时请保留原始版权声明与许可文本。

## 文稿（CC BY-SA 4.0 + SATA） {#content-license}

除特别标注外，所有文稿、插图与说明内容均遵循：

1. [Creative Commons BY-SA 4.0 国际许可协议](https://creativecommons.org/licenses/by-sa/4.0/deed.zh)
2. 叠加的 [Star And Thank Author (SATA) License](https://github.com/zTrix/sata-license)

这意味着：

- **署名**：引用或转载时，请注明“Halo CMS 知识库（MHCGA）”以及原文链接。
- **相同方式共享**：如对内容做出修改、翻译或二次创作，需继续以 CC BY-SA 4.0 提供。
- **不得施加额外限制**：不得加入 DRM、付费墙或与 CC BY-SA 4.0 条款冲突的附加条件。
- **SATA 额外要求**：在分享或使用内容前，请先为 [GitHub 仓库](https://github.com/MHCGA/blog) 点亮一颗 Star，以示感谢。
